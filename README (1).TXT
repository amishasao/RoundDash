PROJECT TITLE:RoundDash
PURPOSE OF PROJECT: Learn how to better program on Mayflower
VERSION or DATE:10
HOW TO START THIS PROJECT:Press C or D depending on whether you want to start as a cat or a dog
AUTHORS:Erek Niemyjski, Amisha Sao, Brandon Nyguen 
USER INSTRUCTIONS: The right arrow key moves the actor right, the left arrow key moves the actor left, the space bar makes the actor move up, and the up arrow makes the actor climb upwards. The objective of the game is to collect as many coins as possible without touching too many of the skulls. As you collect more coins you will move up levels, unlocking better rewards but be careful, as these rewards come with more hazards.
